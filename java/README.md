### 该工程是Tars框架Java语言的源代码 ###


| 目录名称               | 功能               |
| ------------------ | ---------------- |
| net                | 框架rpc网络库         |
| core               | 框架rpc的实现         |
| tools              | 框架工具的实现，maven插件等 |
| examples           | 框架的示例代码          |
| distributedContext | 框架的分布式上下文        |
| protobuf           | 框架的pb协议支持        |
| spring             | 框架的spring相关支持      |
